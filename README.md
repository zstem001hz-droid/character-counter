# Character Counter - State & Events
**Author:** Zac White

## Overview
An interactive character counter application built with React and TypeScript.
Demonstrates state management, event handling, and component communication.

## Tech Stack
- [React 19](https://react.dev/) — UI library
- [TypeScript](https://www.typescriptlang.org/) — Type safety
- [Vite 8](https://vitejs.dev/) — Build tool and dev server

## Components

### TextInput
Handles user text input and reports changes to parent via callback.

### StatsDisplay
Displays real-time text statistics — character count, word count, and reading time.

### CharacterCounter
Parent component that manages state and composes TextInput and StatsDisplay.

## Getting Started
```bash
npm install
npm run dev
```
